Q1: What is the difference between var, let, and const?

answer: 
var -> can be reassigned and redeclared . it is old way. should avoid using it.

let -> can be reassigned but not redeclared . should use it when the value may change.

const -> cannot be reassigned and redeclared. it is used for most of the time. 

Q2: What is the spread operator (...)?

answer: The spread operator expands the elements of an array or object. It copies or expands values

Q3: What is the difference between map(), filter(), and forEach()?

answer:

map() -> it is used when you want to change every element of an array and returns a new array with modified values.

filter() -> it is used when you want some elements using conditions. it also returns new array.

forEach() -> forEach is an array loop. it is used when you want to loop through an array to do something.
